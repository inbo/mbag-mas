This folder contains the following data files related to agricultural plots.
Available at https://landbouwcijfers.vlaanderen.be/open-geodata-landbouwgebruikspercelen

- Shapefiles converted to parquet files
  - Parquet files in pilot study used to calculate crop proportion variables during sampling frame and afterwards
  - For plot size in pilot study the Shapefiles were used
- In MBAG MAS only used to calculate explanatory variables
  - Proportion of crops and plot sizes based on geopackage files

MAS pilot repository: https://github.com/inbo/mas-piloot
MBAG MAS repository: https://github.com/inbo/mbag-mas
